# 问题修复记录

## 修复1：游戏时间流速过快

**日期：** 2026-06-25  
**文件：** `data/components/info.py`  
**行号：** 第300行

### 问题描述
游戏右上角倒计时时间流逝速度约为正常速度的2.5倍，每秒实际减少约2.5秒。

### 原因分析
`update_count_down_clock()` 方法中使用 `pg.time.get_ticks()` 获取毫秒级时间，但阈值设为 `400`：
```python
elif (level_info[c.CURRENT_TIME] - self.current_time) > 400:
```
400毫秒 = 0.4秒减1，导致计时器约2.5倍速运行。

### 修复内容
将阈值从 `400` 改为 `1000`（1秒）：
```python
# 修改前：
elif (level_info[c.CURRENT_TIME] - self.current_time) > 400:

# 修改后：
elif (level_info[c.CURRENT_TIME] - self.current_time) > 1000:
```
